Главное меню программы

Главное меню программу состоит из следующих пунктов:

1. Файл.

2. Статистика.

3. Установки.

4. Окна.

Файл.

 

Пункт файл состоит из двух подпунктов:

 

Вентиляция

При выборе данного пункта на экран выводится окно с изображенной в нем структурной схемой вентиляции. В данном окне отображается состояние ляд и подшипников, а также Выводятся численные значения депрессии, расхода воздуха, температуры воздуха, давления воды, температуры воды.

 

Выход

При выборе данного пункта осуществляется выход из программы (см. Операции осуществляемые при выходе).

 

Статистика.

 

Пункт статистика состоит из трех подпунктов:

 

Самописец

При выборе данного пункта на экран выводится самописец.

 

Параметры БД.

При выборе данного пункта на экран выводится окно с тремя закладками Основная база данных, Аварийная база данных, Резервная база данных. В данном окне отражается содержимое базы данных в виде таблицы, существует возможность вывести базу данных за определенное число, а также вывести определенный параметр. Также в окне присутствуют кнопки просмотр печати и печать базы данных.

 

График

Выводит график изменения выбранных параметров за определенное число.(см. Настройка и печать графика)

 

Установки.

Пункт установки состоит из четырех подпунктов:

 

Установка каналов

 

Выводит окно установки каналов.

Адреса и индексы

 

Выводит окно установки адресов и индексов.

Установка графиков и печати

 

Выводит окно установки графиков и печати.

Отключение сигналов

Выводит окно отключения сигналов предназначение для временного отключения аварийного звука для каждого из аварийных параметров.
