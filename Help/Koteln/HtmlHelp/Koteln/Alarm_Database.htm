Аварийная база данных

Запись в аварийную базу данных осуществляется тогда когда происходит какая-либо аварийная ситуация. Частота записи в аварийную базу данных в несколько раз выше, чем в основную базу данных (см. Основная база данных). Интервал записи можно менять в окне установки параметров баз данных (см. Окно настройки параметров баз данных). Структура и название индексов аварийной базы данных аналогичны основной базе данных.

 

Смотрите также Основная база данных, Резервная база данных.
