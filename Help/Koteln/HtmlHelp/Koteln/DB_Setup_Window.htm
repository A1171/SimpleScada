Окно настройки параметров баз данных

Данное окно вызывается выбором подпункта “Параметры баз данных” в пункте меню статистика главного меню программы. Данное окно предназначено для просмотра и печати баз данных, а также настройки частоты записи в базы данных. Данное окно имеет три закладки с названиями “Стандартная база данных”, “База аварий”, “Резервная база данных”. При выборе одной из закладок вызываются Основная база данных, Аварийная база данных и Резервная база данных соответственно. При выборе одной из баз данных вы можете настроить частоту записи в базу данных, настроить сортировку базы данных, а также осуществить фильтрацию по одному из параметров. Существует возможность просмотреть и напечатать базу данных. При часть базы данных выведенная на печать будет зависеть от сделанных вами настроек. При выборе Резервной базы данных отсутствует возможность настроить частоту записи в базу т.к. резервная база данных является копией основной. Копирование осуществляется при выходе из программы (см. Операции осуществляемые при выходе).
