Окно настройки каналов

Некорректное изменение данных в этом окне может повлечь за собой сбои в работе программы или сделать невозможным ее дальнейшую работу.

При изменении данных строго следуйте нижеприведенным рекомендациям и ограничениям.

 

Данное окно вызывается нажатием подпункта “Установка каналов” в пункте “Установки” главного меню программы. Окно предназначено для установки параметров приема, контроля и отображения принимаемых сигналов по каждому контролируемому каналу. Окно состоит из нескольких групп.

 

Порядковый номер и источник сигнала

 

Данная группа содержит четыре окна.

 

Окно номер

В данном окне отображается порядковый номер принимаемого сигнала который невозможно менять.

 

Окно Адам

В данном окне показан номер адама, с которого снимается сигнал. Диапазон значений 0-10 только целые числа. Изменять данный параметр не рекомендуется, так как данный сигнал может быть связан с визуальным компонентом на мнемосхеме и изменение номера адама может повлечь за собой некорректную работу мнемосхемы.

 

Окно Канал

В данном окне показан номер канала, с которого снимается сигнал. Диапазон значений 0-7 только целые числа. Изменять данный параметр не рекомендуется, так как данный сигнал может быть связан с визуальным компонентом на мнемосхеме и изменение номера адама может повлечь за собой некорректную работу мнемосхемы.

 

Окно Индекс группы

Сопоставляет параметру группе определенный групповой индекс. Диапазон значений 0-50 только целые числа. При значении –1 параметр на имеет индекса.

Более подробно см. Индексы групп.

 

 

Параметр

 

Данная группа используется только для отображения параметра и содержит два окна.

 

Окно Название параметра

 

Определяет название параметра. Длинна названия до 30 символов.

Окно Ед. измерения

 

Определяет единицы измерения. Длинна до 30 символов.

 

Параметры выходного сигнала

 

Содержит шесть окон.

 

Мин. значение

Задает минимальное значение параметра. При более низком значении сигнала выдается предупреждение.

 

Макс. значение

Задает максимальное значение параметра. При более высоком значении сигнала выдается предупреждение.

 

Предуп-реждение о мин.

 

Значение при опускании параметра, ниже которого выдается предупреждение. Должно быть больше или равно Мин. Значению.

Предуп-реждение о макс.

 

Значение при поднятии параметра, выше которого выдается предупреждение. Должно быть меньше или равно Макс. Значению.

Смещение

При выводе параметра к нему добавляется данное значение после умножения на масштаб.

 

Масштаб

При выводе параметра сигнал, полученный с адама, умножается на данное значение.

 

Звуковые файлы

 

Данная группа определяет обработку аварийных ситуаций (см. Обработка аварийных ситуаций).

Окно Звук макс. Предупреждения

Содержит сообщение для обработки ситуации когда значение параметра выше значения содержащегося в окне предупреждение о макс.

 

Окно Звук мин. Предупреждения

Содержит сообщение для обработки ситуации когда значение параметра ниже значения содержащегося в окне предупреждение о мин.

 

Окно Звук верхнего порога

Содержит сообщение для обработки ситуации когда значение параметра выше значения содержащегося в окне макс. Значение.

 

Окно Звук нижнего порога

Содержит сообщение для обработки ситуации когда значение параметра ниже значения содержащегося в окне мин. Значение.

 

 

Смотрите также типы сообщений.

 

В самом низу окна установки каналов присутствуют управляющие кнопки для перемещения по базе данных снимаемых параметров, а также кнопки “OK” для сохранения изменений, “Cansel” для отмены изменений и “Стандартные настройки” для восстановления первоначальных настроек (см. Каталог ReservDB).
