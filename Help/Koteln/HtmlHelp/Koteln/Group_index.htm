Индексы групп

В данной программе существует возможность объединять несколько параметров одним индексом. При объединении, в случае возникновения аварийной ситуации по одному или нескольким параметрам не будет выводится стандартное предупреждение, а выведется сообщение сопоставленное индексу (см. Окно установки адресов и индексов). Данная возможность сделана для разгромождения строки состояния. Например можно сопоставить всем параметрам связанным с давлением один индекс, и данному индексу присвоить сообщение “Давление”. В этом случае при возникновении любой аварии по одному или нескольким параметрам которым сопоставлен данный индекс в строке состояния будет выведено слово давление.

Для объединения параметров индексом необходимо вывести Окно установки каналов нажав подпункт “Установка каналов” в пункте “Установки” главного меню и записать в окошке “Индекс группы” любое число больше нуля для всех необходимых параметров. После этого, необходимо вызвать окно установки адресов и индексов нажав подпункт “Адреса и индексы” в пункте “Установки” главного меню выбрать необходимый индекс и записать сообщение.
