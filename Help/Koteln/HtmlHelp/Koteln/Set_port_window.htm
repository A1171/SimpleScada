Окно настройки портов и индексов

_ Данное окно вызывается выбором подпункта “Адреса и индексы” в пункте “Установки” главного меню программы. Окно предназначено для сопоставления адаму адреса и сопоставления индексу сообщения.

Окно содержит две группы.

Установка адреса

Данная группа служит для изменения адреса по которому программа обращается к адаму и содержит два элемента.

Окно номера адама.

В данном окне прокручиваются порядковые номера адамов используемые при установке параметров каналов (см. Окно установки каналов).

 

Окно адреса

В данном окне записывается шестнадцатеричный адрес из двух символов с использованием цифр и заглавных букв (A B C D E F) латинского алфавита. При отсутствии записи или записи nn адрес опрашиваться не будет.

 

При установке несуществующего адреса адама для любого номера адама программа не сможет с ним связаться, и будет выдавать аварийные сообщения.

 

Установка индексов

 

Данная группа предназначена для сопоставления индексам аварийных сообщений (см. Индексы групп). Окно состоит из двух элементов.

 

Окно Номер индекса

 

В данном окне прокручиваются номера индексов.

Окно сообщение

 

В данном окне записываются аварийные сообщения для каждого индекса.
