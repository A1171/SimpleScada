Настройка и запуск программы после установки Windows

Для запуска после переустановки Windows необходимо сделать следующее:

1. Установить утилиты Database Engine и Database Desktop.

2. Скопировать программу в каталог в котором она в дальнейшем будет находиться.

3. Создать в утилите Database Engine новый псевдоним с названием Adams и при выходе из утилиты подтвердить изменения.

4. В качестве пути псевдонима записать путь до каталога, в который скопирована программа.

5. Скопировать в каталог Windows\System\ файлы

 vcl40.bpl

 vclx40.bpl

 vcldb40.bpl

 qrpt40.bpl

6. Создать в главном меню и на рабочем столе ярлыки для файла Adamcont.exe если это необходимо.

7. Запустить программу напрямую или с помощью ярлыка.
