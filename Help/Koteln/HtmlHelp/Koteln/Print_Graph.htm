Настройка и печать графика

Окно графика состоит из области отображения графика, панели управления и выдвижной шкалы. Область отображения служит для вывода выбранных графиков. Панель управления служит для выбора графиков и их параметров и содержит следующие управляющие элементы:

 

Кнопка добавить график

При нажатии на экране появится окно выбора параметров. Для добавления графика выберете параметр и нажмите OK. Будет предложено окно выбора цвета. Выберите цвет и нажмите OK.

 

Кнопка удалить график

 

Выводит на экран список графиков.

Полоса прокрутки масштаба графика

Служит для изменения масштаба графика по оси времени. Для применения изменений нажмите кнопку “Применить изменения.”

 

Окно выбора даты.

Служит для ввода даты за которую отображен график. После введения новой даты нажмите кнопку “Применить изменения.”

Кнопка “Применить изменения”.

Служит для применения изменений после изменения масштаба по оси времени.

 

Выдвижная шкала служит для вывода шкалы значений для любого параметра график которого выбран. Данная шкала появляется при подводе курсора к левой границе окна графика. Для того чтобы выбрать график для которого показан масштаб необходимо щелкнуть на выдвижной шкале правой кнопкой мыши список графиков содержащий все выведенные графики и их цвета. После выбора графика необходимо нажать кнопку OK. Для удаления выдвижной шкалы с экрана необходимо щелкнуть на ней левой кнопкой мыши.
