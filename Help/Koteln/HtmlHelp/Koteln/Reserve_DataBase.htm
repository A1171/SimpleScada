Резервная база данных

Резервная база данных хранит копию основной базы данных за последние два дня. Просмотреть данную БД можно из окна настройки баз данных. Обновление данной базы данных осуществляется при выходе из программы (см. Операции осуществляемые при выходе). Название и структура индексов данной базы аналогична Основной базе данных.

 

Смотрите также Основная база данных, Аварийная база данных.
