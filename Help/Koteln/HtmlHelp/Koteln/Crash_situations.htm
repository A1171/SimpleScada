Обработка аварийных ситуаций

Для каждого параметра контролируемого системой существует возможность задать диапазон допустимых значений окне установки каналов (см. Окно установки каналов). При уходе любого параметра из допустимого диапазона значений возникает аварийная ситуация. В данной программе существует возможность включения или отключения контроля аварийных ситуаций, а также настройка выдаваемого в случае аварии сообщения и установка звука проигрываемого при возникновении аварийных событий.

Для каждого параметра в программе предусмотрено четыре типа аварийных ситуаций.

1. Значение параметра выше максимально допустимого значения.

2. Значение параметра ниже минимально допустимого значения.

3. Значение зашкалило за установленное максимальное значение.

4. Значение параметра ушло ниже минимального значения.

Ситуация когда возникают эти события показана на рисунке.

При заходе параметра в первую зону возникает первое событие, при заходе за верхнюю границу 3 событие. При уходе во вторую зону второе событие, при уходе за нижнюю границу 4 событие. При установке каждого параметра в окне установки каналов (см. Окно установки каналов) существует четыре окна в каждое из которых необходимо вписать специальное сообщение (см. Типы сообщений.) или ни чего не вписывать, в этом случае аварийная ситуация ни как не будет контролироваться.
