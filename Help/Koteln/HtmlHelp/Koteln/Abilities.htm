Возможности программы

Программа реализует следующие функции:

1. Отображает на экране состояние ляд.

2. Сигнализирует о перегреве подшипников.

3. Выводит значения депрессии, расхода воздуха, температуры воздуха, давления воды, температуры воды.

4. Выдает звуковые сигналы и сообщения о выходе параметров из допустимого диапазона (см. Звук).

5. Выводит самописец для любого параметра.

6. Позволяет делать настройку параметров.

7. Ведет запись состояния системы в базу данных (см. Основная база данных).

8. Записывает аварийную базу данных при выходе параметров из допустимого диапазона.

9. В 1400 программа сохраняет резервную базу данных за последние два дня.

10. Печатает отчеты по базам данных в виде графиков и таблиц.
