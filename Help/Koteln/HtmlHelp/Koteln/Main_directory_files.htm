Файлы содержащиеся в основном каталоге

В основном каталоге содержатся следующие файлы:

 

AdamCont.exe

 

Основной файл программы.

Settings.ini

 

Главный файл настроек программы.

SetAdres.ini

 

Файл настроек адресов адамов.

SetGraph.ini

 

Файл настройки графиков.

Adams3.db

 

БД параметров.

Adams3.px, Adams3.val

 

Основной индекс БД параметров.

Alarms.db

 

Аварийная база данных.

Alarms.px, Alarms.val

 

Основной индекс аварийной БД.

Alarms.XG0, Alarms.YG0,

Alarms.XG1, Alarms.YG1. 

Два дополнительных индекса Аварийной БД.

Data1.db

 

Основная база данных статистического учета изменения параметров.

 

Data1.px, Data1.val

 

Основной индекс БД статистического учета параметров.

Data1.XG0, Data1.YG0,

Data1.XG1, Data1.YG1. 

Два дополнительных индекса Главной БД.

ReserDat.db

 

Резервная база данных.

ReserDat.px, ReserDat.val

 

Основной индекс резервной БД.

ReserDat.XG0, ReserDat.YG0,

ReserDat.XG1, ReserDat.YG1. 

Два дополнительных индекса резервной БД.
