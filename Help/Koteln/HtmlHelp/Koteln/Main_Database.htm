Основная база данных

В основной базе данных хранятся значения всех параметров за последние два месяца. При каждом выходе из программы удаляются все записи с датой меньшей чем текущая дата минус 60 дней. Данная база данных записывается автоматически с интервалом установленным в окне установки параметров баз данных (см. Окно настройки параметров баз данных). Данный интервал можно изменять, но отключить запись базы данных невозможно.

В 1400 осуществляется копирование данных за последние два дня из основной базы данных в резервную. Просмотр основной и аварийной базы данных и удаление устаревших записей. Время затрачиваемое на просмотр основной и аварийной базы данных и удаление устаревших записей зависит от быстродействия компьютера и количества устаревших записей. Иногда эта операция может занимать до 10 минут. При этом опрос адамов сильно замедляется.

 

Данная база данных имеет три индекса используемые для разных случаев сортировки базы данных.

Основной индекс включает все поля базы данных.
