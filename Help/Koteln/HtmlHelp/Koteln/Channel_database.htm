База каналов

В данной базе данных содержатся описания всех параметров контролируемых системой. В соответствии с этой базой осуществляется опрос адамов, проверка вхождения в допустимый диапазон каждой величины, расчет масштабов графиков. Редактирование данной базы данных осуществляется с помощью окна установки каналов (см. Окно установки каналов). В случае неправильного изменения содержимого данной базы данных программа может стать не работоспособной. В этом случае базу данных следует восстановить из резервной копии, или отредактировать с помощью утилиты Database Desktop . Данная база данных имеет один основной индекс включающий поле “Number”. В случае потери индекса его можно восстановить с помощью утилиты Database Desktop, но проще скопировать базу данных из резерва. База каналов не изменяется программой. Изменения может вносить только пользователь.
