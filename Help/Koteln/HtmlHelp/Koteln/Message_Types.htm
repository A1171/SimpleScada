Типы сообщений

Любое сообщение об обработке аварийной ситуации может быть трех типов.

1. Имя звукового wav файла в этом случае будет проигран звук и выведено сообщение. Wav файл должен находится в подкаталоге Wav основного каталога программы.

2. Слово Yes в этом случае звука не будет но будет выводится сообщение об аварии.

3. В случае если сообщение отсутствует аварийная ситуация не будет обрабатываться.
